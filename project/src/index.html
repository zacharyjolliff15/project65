Problem 2: The BankAccount Class

Objective: Create a class with attributes and methods, using a constructor for initial setup.

Task:

  • Define a class named BankAccount.
  • Give the class attributes like accountNumber (string or integer), accountHolderName (string), and balance (float or decimal).
  • Create a constructor that accepts the accountNumber and accountHolderName as arguments. The constructor should initialize these attributes and set the initial balance to 0.0.
  • Add two methods to the class:
    • deposit(amount): Adds the specified amount to the balance. Ensure the amount is positive.
    • withdraw(amount): Subtracts the specified amount from the balance. Ensure the amount is positive and that there are sufficient funds before withdrawing. Return true if successful, false otherwise.
  • Add two more methods to the class:
    • getBalance(): Returns the current balance.
    • displayAccountInfo(): Prints the account holder's name, account number, and current balance.
  • Instantiate a BankAccount object using your constructor.
  • Perform a sequence of operations: deposit some money, try to withdraw a valid amount, try to withdraw an amount greater than the balance, and finally display the account information.
